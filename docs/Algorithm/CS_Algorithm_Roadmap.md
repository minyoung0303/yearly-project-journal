# 🗺️ CS 기술면접 대비 알고리즘 · 자료구조 로드맵 (6주)

## 📌 학습 원칙

- ❌ 문제 수 늘리기 → ⭕ **적게 풀고 깊게 이해하기**
- 모든 개념은 아래 4가지를 함께 정리

  1. 정의
  2. 언제 사용하는가
  3. 시간복잡도
  4. 실제 사용 사례

- 모든 알고리즘 문제는 **접근 과정 + 선택 이유**를 설명할 수 있어야 함

---

# 🟦 1주차: 자료구조 기초

## 🎯 학습 목표

- 기본 자료구조를 설명할 수 있다
- 배열/리스트 차이를 명확히 말할 수 있다

## 📚 이론 학습

- Array vs List
- Stack
- Queue
- Deque
- Hash Table (Dictionary)
- Set

## 🧠 면접 핵심 질문

- 배열과 리스트의 차이는?
- 스택과 큐는 언제 사용하나요?
- 해시테이블의 평균 시간복잡도는 왜 O(1)인가요?

## 🧩 알고리즘 유형

- 배열 순회
- 문자열 처리
- 스택 활용 문제

## ✅ 추천 문제 예시

- 숫자 문자열에서 숫자 합 구하기
- 괄호 검사
- 짝지어 제거하기

# 🟦 2주차: 트리 & 힙

## 🎯 학습 목표

- 트리 구조를 그림으로 설명할 수 있다
- DFS/BFS 차이를 말할 수 있다

## 📚 이론 학습

- Tree 기본 개념
- Binary Tree
- Binary Search Tree (BST)
- Heap / Priority Queue

## 🧠 면접 핵심 질문

- 이진트리와 BST의 차이는?
- 힙은 왜 정렬 대신 사용하는가?
- DFS와 BFS의 차이는?

## 🧩 알고리즘 유형

- 트리 순회
- DFS / BFS

## ✅ 추천 문제 예시

- 트리 깊이 구하기
- 같은 레벨 노드 출력
- 우선순위 큐 활용 문제

# 🟦 3주차: 정렬 & 탐색

## 🎯 학습 목표

- 정렬 알고리즘을 비교 설명할 수 있다
- 이분 탐색 조건을 정확히 이해한다

## 📚 이론 학습

- Bubble / Selection / Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort
- Binary Search

## 🧠 면접 핵심 질문

- 각 정렬 알고리즘의 시간복잡도 비교
- 안정 정렬이란?
- 이분 탐색은 언제 사용 가능한가?

## 🧩 알고리즘 유형

- 정렬 구현
- 이분 탐색 응용 문제

## ✅ 추천 문제 예시

- K번째 수
- 예산
- 입국 심사

# 🟦 4주차: 그래프

## 🎯 학습 목표

- 그래프 표현 방식을 설명할 수 있다
- 최단 경로 문제 접근을 말할 수 있다

## 📚 이론 학습

- Graph 개념
- 인접 리스트 vs 인접 행렬
- 방향 그래프 / 무방향 그래프

## 🧩 알고리즘

- DFS / BFS
- Dijkstra
- Union-Find

## 🧠 면접 핵심 질문

- DFS와 BFS 중 어떤 걸 선택할 기준은?
- 다익스트라는 왜 음수 간선이 안되나요?

## ✅ 추천 문제 예시

- 네트워크
- 가장 먼 노드
- 섬 개수 세기

# 🟦 6주차: 실전 면접 대비

## 🎯 학습 목표

- 문제를 말로 풀 수 있다
- 화이트보드 코딩에 익숙해진다

## 📚 학습 내용

- 빈출 알고리즘 문제 반복
- 자료구조 개념 요약 정리
- 실전 면접 질문 대비

## 🎤 실전 연습

- 문제 접근 설명 → 코드 작성 → 시간복잡도 분석
- 자료구조 선택 이유 설명

## ✅ 최종 체크리스트

- 자료구조별 시간복잡도 암기
- DFS/BFS/DP 접근 흐름 설명 가능
- 최소 1문제는 **말로만 설명 가능**

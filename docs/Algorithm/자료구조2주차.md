> 목표: **트리 구조를 그림으로 설명하고, DFS/BFS를 말로 비교 설명할 수 있다**

---

## 🎯 학습 목표

- 트리 구조를 말과 그림으로 설명할 수 있다
- 이진트리, BST, 힙의 차이를 명확히 말할 수 있다
- DFS / BFS 차이를 사용 사례와 함께 설명할 수 있다

---

## 📚 이론 학습

### 1️⃣ Tree 기본 개념

#### ✅ 정의

- **계층적 자료구조**
- 노드(Node)와 간선(Edge)으로 구성
- **사이클이 없는 그래프**

#### ✅ 주요 용어

- Root: 최상위 노드
- Parent / Child
- Leaf: 자식이 없는 노드
- Depth / Height

#### 🧠 핵심 한 줄

> 트리는 **계층 구조를 표현**하기 위한 자료구조

---

### 2️⃣ Binary Tree (이진 트리)

#### ✅ 정의

- 각 노드가 **최대 2개의 자식**을 가지는 트리

#### ✅ 특징

- 왼쪽/오른쪽 자식의 **값 크기 규칙은 없음**
- 구조 자체에 의미

#### 📌 사용 예

- 트리 순회 연습
- 표현 구조가 중요한 경우

---

### 3️⃣ Binary Search Tree (BST)

#### ✅ 정의

- 이진 트리의 한 종류

#### ✅ 규칙

- 왼쪽 서브트리 < 부모 노드 < 오른쪽 서브트리

#### ✅ 시간복잡도

- 탐색 / 삽입 / 삭제: 평균 **O(log N)**
- 최악(편향 트리): **O(N)**

#### 📌 사용 예

- 정렬된 데이터 탐색
- 범위 검색

#### 🧠 핵심 한 줄

> BST는 **탐색을 빠르게 하기 위한 트리**

---

### 4️⃣ Heap / Priority Queue

#### ✅ 정의

- **완전 이진 트리 기반 자료구조**
- 우선순위가 가장 높은 값이 항상 루트

#### ✅ 종류

- Max Heap: 부모 ≥ 자식
- Min Heap: 부모 ≤ 자식

#### ✅ 시간복잡도

- 삽입 / 삭제: **O(log N)**
- 최대/최소 조회: **O(1)**

#### 📌 사용 예

- 우선순위 큐
- 스케줄링
- 다익스트라 알고리즘

#### 🧠 핵심 한 줄

> 힙은 **정렬하지 않고도 최댓값/최솟값을 빠르게 얻기 위한 구조**

---

## 🧠 DFS / BFS

### 5️⃣ DFS (Depth First Search)

#### ✅ 특징

- 깊이 우선 탐색
- 스택 or 재귀 사용

#### 📌 사용 예

- 모든 경로 탐색
- 백트래킹

---

### 6️⃣ BFS (Breadth First Search)

#### ✅ 특징

- 너비 우선 탐색
- 큐 사용

#### 📌 사용 예

- 최단 거리 탐색
- 레벨별 탐색

---

### 🔍 DFS vs BFS 비교

| 구분      | DFS         | BFS           |
| --------- | ----------- | ------------- |
| 탐색 방식 | 깊이 우선   | 너비 우선     |
| 자료구조  | 스택        | 큐            |
| 장점      | 메모리 적음 | 최단거리 보장 |
| 사용처    | 경로 탐색   | 거리/레벨     |

---

## 🎤 기술면접 핵심 질문 & 답변

### 🎙️ Q1. 이진트리와 BST의 차이는?

> 이진트리는 각 노드가 최대 두 개의 자식을 가지는 트리 구조 자체를 의미하고, 값의 크기에 대한 규칙은 없습니다.
> 반면 BST는 왼쪽 자식은 부모보다 작고 오른쪽 자식은 부모보다 크다는 규칙이 있어 탐색을 빠르게 하기 위한 트리입니다.

---

### 🎙️ Q2. 힙은 왜 정렬 대신 사용하나요?

> 힙은 전체를 정렬하지 않아도 최댓값이나 최솟값을 O(1)에 조회할 수 있기 때문입니다.
> 정렬은 O(N log N)이 필요하지만, 힙은 삽입과 삭제가 O(log N)으로 효율적입니다.

---

### 🎙️ Q3. DFS와 BFS의 차이는?

> DFS는 한 경로를 끝까지 탐색하는 방식으로 모든 경우를 탐색하거나 백트래킹 문제에 적합하고,
> BFS는 레벨 단위로 탐색하여 최단 거리나 최소 횟수를 구하는 문제에 적합합니다.
